1. Đăng ký tại https://wakatime.com
2. Cài WakaTime extension trong VS Code
3. wakatime.com/settings/account → copy API Key
4. GitHub repo → Settings → Secrets → New secret
   Name: WAKATIME_API_KEY  |  Value: (paste API key)
5. Tạo file .github/workflows/waka-readme.yml với nội dung:

name: Waka Readme
on:
  schedule:
    - cron: '0 0 * * *'
  workflow_dispatch:
jobs:
  update-readme:
    runs-on: ubuntu-latest
    steps:
      - uses: anmol098/waka-readme-stats@master
        with:
          WAKATIME_API_KEY: ${{ secrets.WAKATIME_API_KEY }}
          GH_TOKEN: ${{ secrets.GH_TOKEN }}
          SHOW_LANGUAGE: "True"
          SHOW_PROJECTS: "True"
